===============================
printer73x documentation README
===============================

Use Sphinx (http://sphinx.pocoo.org) to build the HTML documentation.
You'll need the following 3rd party extensions:

- numpydoc (http://pypi.python.org/pypi/numpydoc)

- ipython_console_highlighting (found in Sphinx extensions directory on
  documentation directory of distribution of project IPython
  [https://github.com/ipython/ipython/blob/master/docs/sphinxext/ipython_console_highlighting.py for example])

You can find the HTML documentation online at the project's site
http://www.dreilopz.me/printer73x.
